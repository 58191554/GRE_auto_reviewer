# GRE-AutoReviewer

自动复习单词，加入的单词将自动保存在words.md. 

Review-by-Importance公式：
$$
I=\frac{T_{max}-T_{last\_view}}{T_{max}-T_{min}}+(1-accuracy)
$$
 

| Operation | Function |
| --------- | -------- |
| `A`       | 加入单词 |
| `Q`       | 退出·    |
| `R`       | 复习     |
| `rst`     | 重置     |

可以在history.md中查看每天的学习量，包括新的加入的单词，和今天复习的单词
